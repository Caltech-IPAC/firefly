Getting more Help on Using the Spitzer Heritage Archive

More information on Spitzer's cryogenic mission can be found at IRSA's Spitzer site. A printable user manual for the Spitzer Heritage Archive is available at this site as well.

More information on Spitzer's current mission can be found at http://ssc.spitzer.caltech.edu.

You can submit questions to the IRSA helpdesk.

A set of frequently asked questions (FAQs) is here.

You can also access the SHA from an application program interface (API). See that page, or the PDF SHA manual (available at the IRSA Spitzer site), for more information.

Found a bug? The known bugs and issues in this version of the Spitzer Heritage Archive (SHA) are listed here. If you think you have found a bug, before reporting it, please check this list, and read the online SHA help. It may be a "feature" we already know about. If you have found a new, real bug then please do contact us via the IRSA helpdesk. Please include your operating system version and your browser software and version. If you can, please also include any specific error message you may have gotten. (NB: In our testing, copying shortcuts worked on Windows and Linux; the openapple-C did not work on Macs, but selecting and clicking the right mouse button did.)
