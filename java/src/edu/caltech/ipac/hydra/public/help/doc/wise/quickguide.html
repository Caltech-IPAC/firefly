Searching: Many search methods are available. Most people will probably want to search by position, which is the default option. Target names can be automatically resolved into coordinates using NED or SIMBAD, or you can specify coordinates. See the page on searching for more information.

Hints, Tips, and Term Definitions:

  • Source Catalog Queries: In addition to searching for WISE images, various WISE multiband source lists are also available for queries. Catalog queries are most easily performed using the separate IRSA catalog search tool. However, as part of the image service, WISE catalog sources within previously displayed WISE images may be retrieved using the Catalogs tab on the top toolbar. Please consult the page on Catalogs for more information.

  • Search Results: The search results appear in up to several tabs: Single Exposure (Level 1b), Atlas (Level 3), and a variety of potential catalog searches. Each tab provides a different look at the data retrieved by your search. See the page on understanding your search results for more information.

  • Filtering: Filters, either imposed on your initial search or via your results page, can provide a powerful tool for narrowing down the search results to the observations you want.

  • Visualizing: Visualization is a powerful tool for understanding how your object of interest was observed by WISE and deciding which data to download.

  • Downloading Data: You can select specific datasets by clicking on their respective checkboxes, then click "Prepare Download" to begin packaging your data for download in the background. See the page on the Background Monitor for more information.
