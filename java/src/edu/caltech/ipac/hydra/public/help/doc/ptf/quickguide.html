IRSA/PTF Image Service Quick Start Guide: Getting Up and Running FAST

Searching: Many search methods are available. Most people will probably want to search by position, which is the default option. Target names can be automatically resolved into coordinates using NED or SIMBAD, or you can specify coordinates. Cone searching is the only option at this time. See the page on searching for more information.
